Your personal library keeps all your saved book summaries in one place for easy access and reference.

## Saving summaries

<Steps>
  <Step title="Sign in to your account">
    You need to be authenticated to save summaries to your library.
    
    - If you're not signed in, you'll see a prompt on the summary page
    - Click Sign In and authenticate
    - Return to the summary page after authentication
  </Step>

  <Step title="Save from the summary page">
    On any summary page:
    
    1. Look for the bookmark or save icon in the header
    2. Click to add the summary to your library
    3. The icon changes to indicate it's saved
  </Step>

  <Step title="Access your library">
    Navigate to your library from the main navigation menu to see all saved summaries.
  </Step>
</Steps>

## Library features

### Summary cards

Each saved book appears as a card showing:

- **Book title and author** - Quick identification
- **Core thesis preview** - First two lines of the main argument
- **Chapter count** - How many chapters were summarized
- **Save date** - When you added it to your library ("Today", "Yesterday", etc.)
- **Quick actions** - Chat, View, or Remove buttons

### Quick actions

From your library, you can:

- **Chat** - Start a new chat session with the book
- **View** - Open the full summary page
- **Remove** - Remove the book from your library (with confirmation)

<Note>
Removing a book from your library doesn't delete the summary. You can still access it if you have the direct link.
</Note>

### Empty library state

If you haven't saved any summaries yet:

- You'll see a helpful empty state with instructions
- A prominent button takes you to upload your first book
- Once you save summaries, they appear immediately

## Organizing summaries

Currently, summaries in your library are displayed chronologically with the most recently saved books first. The library shows:

- Total count of saved books
- When each book was saved (relative dates)
- Quick filtering by viewing the book details

## API endpoints

### Save a summary

Add a summary to your library:

```bash
POST /api/summary/{summary_id}/save
Authorization: Bearer <session-token>
```

**Response:**

```json
{
  "saved": true
}
```

### Unsave a summary

Remove a summary from your library:

```bash
DELETE /api/summary/{summary_id}/save
Authorization: Bearer <session-token>
```

**Response:**

```json
{
  "saved": false
}
```

### Get saved summaries

Retrieve all summaries in your library:

```bash
GET /api/library
Authorization: Bearer <session-token>
```

**Response:**

```json
{
  "summaries": [
    {
      "id": "abc123",
      "metadata": {
        "title": "Book Title",
        "author": "Author Name",
        "language": "en",
        "isAutoGenerated": true
      },
      "summary": {
        "coreThesis": "Main argument...",
        "keyInsights": ["Insight 1", "Insight 2"],
        "actionableTakeaways": ["Action 1"],
        "readThisIf": "You should read this if..."
      },
      "chapters": [
        {
          "title": "Chapter 1",
          "summary": "Summary..."
        }
      ],
      "isSaved": true,
      "createdAt": "2026-03-01T12:00:00Z"
    }
  ]
}
```

### Check if summary is saved

When retrieving a summary by ID, the response includes an `isSaved` field:

```bash
GET /api/summary/{summary_id}
Authorization: Bearer <session-token>
```

The response includes:

```json
{
  "id": "abc123",
  "isSaved": true,
  // ... other fields
}
```

<Tip>
The `isSaved` field is determined based on the authenticated user. Unauthenticated requests always show `false`.
</Tip>

## Exporting summaries

You can export any summary as a PDF:

1. Open the summary page
2. Look for the download or export button
3. The PDF includes the full summary with all sections

**API endpoint:**

```bash
GET /api/summary/{summary_id}/pdf
```

Returns a PDF file with a formatted version of the summary.

## Library limits

Currently, there are no limits on:

- Number of summaries you can save
- How long summaries remain in your library
- Number of times you can access a saved summary

<Warning>
Deleting your account will remove access to your library, but saved summaries remain in the system and can still be accessed via direct links.
</Warning>
