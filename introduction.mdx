# Welcome to Reading Assistant

Reading Assistant is an AI-powered tool that converts books (PDF, EPUB, or plain text) into structured summaries and interactive chat experiences. Extract core ideas from any book quickly, without needing to read it cover to cover.

## Tech stack

Reading Assistant is built with modern technologies:

### Backend
- **Python 3.12** with FastAPI for high-performance APIs
- **SQLAlchemy** ORM with SQLite (configurable to PostgreSQL)
- **ChromaDB** vector store for RAG capabilities
- **LlamaIndex** for document processing and retrieval

### Frontend
- **Next.js 16** with App Router
- **React 19** with TypeScript
- **Tailwind CSS** for styling
- **Better Auth** for authentication

### AI & Infrastructure
- **OpenRouter API** with Gemini 3 Flash for summarization
- **OpenAI embeddings** for semantic search
- **RAG pipeline** for contextual chat responses

<Note>
  Reading Assistant requires API keys for OpenRouter and OpenAI. See the [installation guide](/installation) for details on obtaining and configuring these keys.
</Note>
